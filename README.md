# Gittensor Base Miner — Dashboard

Static GitHub Pages leaderboard for the [gittensor-base-miner](https://github.com/PunchTheDev/gittensor-base-miner) benchmark.

## What this is

A live, public leaderboard showing who holds the top score on the rotating base-miner benchmark. The dashboard reads from `data.json`, which is regenerated by CI in the benchmark repo on each scored submission.

## Structure

```
index.html    — single-page dashboard (leaderboard, pool stats, problem browser)
data.json     — generated data file (leaderboard + problem metadata)
```

## Updating data

Run the data generation script in the benchmark repo:

```bash
cd ../gittensor-base-miner
python scripts/generate_dashboard_data.py
cp dashboard_data.json ../gittensor-miner-dashboard/data.json
```

Then commit and push to `main` — GitHub Pages deploys automatically.

## Deploying

GitHub Pages is configured to serve from the `main` branch root. Enable it under Settings → Pages → Source: Deploy from branch `main` / `/ (root)`.
